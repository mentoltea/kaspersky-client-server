# Сборка
```bash
mkdir build && cd build
cmake ..
cmake --build .
```

По итогу сборки в папке `build/bin` собираются 4 исполняемых файла:
- `client` - клиент
- `server` - сервер
- `handler` - вспомогательный исполняемый файл сервера для создания процесса для обработки клиента
- `statistic` - утилита сбора статистики

# Поддержка
Пока что поддержка только для Windows (кроме MSVC, поскольку MSVC неправильно обрабатывает флаги), однако все модули были сделаны с рассчётом возможности добавления поддержки других платформ (в первую очередь Linux и Unix-подобных систем): в комплексных модулях через концепт *Pointer to implementation* ([tcp](./tcp/), [fifo](./fifo/), [shmem](./shmem/)), в более простых / легче переносимых через простые `#ifdef`.
